<h2>Installation Steps:</h2>

<p>1. Install the dependencies</p>

```
pip install -r ./requirement.txt
```

<p>2. Start the web service</p>

```
python ludotheque/manage.py runserver
```
<p>3. Access on your web brother and enjoy</p>

```
localhost:8000
```
